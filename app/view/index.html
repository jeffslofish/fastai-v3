Classify Programming Lanuage Snippets

Enter some code and the programming language of the code will be displayed!

I can distinguish between the following languages:
bash, c, c#, c++, css, haskell, html, java, javascript, lua, markdown, objective-c, perl, php, python, r, ruby, scala, sql, swift, and vb.net
